# How to Use Linux Ui

::: warning

This manual was tested with Ubuntu 22.04.

:::

1. Navigate to Rainbow-Robotics Web. (https://www.rainbow-robotics.com)
   
2. Access ‘Tech support’ -> ‘Cobot Tech Resources’ from the top bar.

![missing](/technical_docs/common/ui_linux/a1.png)

3. Access the One-Drive link by clicking ‘Download’ from the top bar.  

![missing](/technical_docs/common/ui_linux/a2.png)

4.	Download .zip file from ‘1_Software_Release’ -> ‘1.1_LTS’ -> ‘3_Linux’

5.	Extract .zip file.

6.	Right click the executable file and open Properties.

![missing](/technical_docs/common/ui_linux/a3.png)

7. At Permissions tap, check ‘Allow executing file as program’

![missing](/technical_docs/common/ui_linux/a4.png)

8. Double click the executable file to run the Rainbow Robotics UI.
