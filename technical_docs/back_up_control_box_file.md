# How To Back-up Control Box Files v7.7

### Back-up Control-Box files

::: warning
Please ensure that the control box and the RB UI are connected via **USB**, as backup is only supported under this connection.
:::


![missing](/technical_docs/common/backup/2_1.png)
![missing](/technical_docs/common/backup/2_2.jpg)

When you click "Backup Control-Box files", the files will be saved to the path shown in the popup window. After connecting the tablet to a PC, you can access the saved files by navigating to the specified path. 
![missing](/technical_docs/common/backup/2_3.png)

### Location of the back-up files

Enter **Tablet -> Android -> data**
![missing](/technical_docs/common/backup/1_4.png)

The backup files are stored in the **com.rainbow.cobot -> files** directory. Please refer to the backup date to identify the correct folder. <br>
This folder contains log files, wsl files.
![missing](/technical_docs/common/backup/1_5.png)
