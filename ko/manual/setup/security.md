# Set-up (Security)

### Security 
<br>

#### 기능 설명
각 메뉴 화면의 비밀번호를 설정하거나 화면 켜짐에 대한 설정을 할 수 있습니다.

![missing](/manual/ko/setup/security.png)

1. UI에서 다룰 수 있는 화면마다 비밀번호를 지정할 수 있습니다.
2. - **화면 항상 켜짐** : Make와 Play를 사용할 때 화면을 항상 켤 수 있는 기능입니다.<br>
     (화면이 꺼진 상태에서 연결 끊김을 방지할 수 있으나, 배터리의 소모가 심하기 때문에 해당 내용을 고려하여 사용해야 합니다.)<br>
   - **제스처 모드** : 해당 기능을 켜 놓을 경우 위/아래 제스처를 통해 화면 이동이 가능합니다.

#### 암호 설정 방법

![missing](/manual/common/security/1.png)

1. 초기 비밀번호는 0000 입니다. 초기 비밀번호와 변경 하실 번호를 입력하신 후 하단의 저장을 누른 후 별다른 팝업창이 나타나지 않으면, 비밀번호 세팅이 완료 된 상태입니다. 

![missing](/manual/common/security/2.png)

2. 만약, 위 사진과 같은 팝업이 나타난다면 기존 비밀번호가 잘못된 상태입니다. 

![missing](/manual/common/security/3.png)

3. 자동 로그인을 해제 한 후, 현재 비밀번호를 입력하고 새로운 비밀번호를 2회 입력 후에 저장 버튼을 클릭하면 비밀번호 설정이 완료됩니다. 

#### 화면 항상 켜짐

![missing](/manual/common/security/4.png)

  화면 항상 켜짐을 “사용” 옵션을 선택하면, Make와 Play를 사용할 때 화면을 항상 켤 수 있는 기능입니다. ( 화면이 꺼진 상태에서 연결 끊김을 방지할 수 있으나, 베터리의 소모가 심하기 때문에 해당 내용을 고려하여 사용해야 합니다. )

#### 제스처 모드

![missing](/manual/common/security/5.png)

제스처 모드를 켜놓을 경우 위/아래 제스처를 통해 Make-Play-Setup 화면 이동이 가능합니다. 

### IP List
<br>

![missing](/manual/common/security/6.png)

IP List는 간편하게 제어박스 IP 주소를 여러 개 미리 등록해놓는 기능 입니다. 

![missing](/manual/common/security/7.png)

![missing](/manual/common/security/8.png)

![missing](/manual/common/security/9.png)

연결 페이지에서 IP List를 확인할 수 있습니다. <br> IP 주소란 옆에 화살표 버튼을 클릭하면 IP List에 등록해준 IP로 손쉽게 변경이 가능합니다.
