# 제어박스 내부 파일 백업하기 v7.7

### 제어박스 내부 파일 백업

::: warning
백업은 제어박스와 Ui가 USB 연결이어야 가능합니다. 
:::


![missing](/technical_docs/common/backup/1.png)
![missing](/technical_docs/common/backup/2.png)

제어박스 내부 파일 백업을 클릭하면, 팝업창에 뜬 경로에 저장됩니다. 태블릿과 PC를 연결하신 후 해당 경로로 저장된 파일들을 확인하면 됩니다. 
![missing](/technical_docs/common/backup/1_3.png)

### 백업된 파일 위치

**Tablet -> Android -> data** 를 들어갑니다. 
![missing](/technical_docs/common/backup/1_4.png)

**com.rainbow.cobot -> files** 폴더 안에 백업 받은 파일이 생성되어 있습니다. 백업 받은 날짜를 참고하시어 폴더명을 확인해주세요. <br>
해당 폴더 안에 로그와 wsl 파일이 담겨있습니다. 
![missing](/technical_docs/common/backup/1_5.png)
