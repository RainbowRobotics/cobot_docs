# 제어박스 시간 설정하기 v7.8

:::tip

제어박스의 시간대가 올바르게 설정되지 않은 경우, 이후 로그 확인 시 로그 데이터의 시간 정보가 실제 시간과 불일치할 수 있습니다. 따라서, 제어박스의 시간대를 화면 우측 상단에 표시되는 현재 시간에 맞춰 정확히 설정할 것을 권장합니다.

:::


### 1. 태블릿을 제어박스와 연결하십시오.

### 2. Set-up > Log/Utility 경로로 이동하십시오.

### 3. 제어박스의 시간대를 확인하십시오.

제어박스의 시간대를 확인하십시오. (시간대 설정은 로그 파일의 정확한 이름 지정 및 향후 로봇 유지보수에 영향을 미칩니다.)

![missing](/technical_docs/common/set_cb_time_zone/3.png)

### 4. System > Configuration 경로로 이동하십시오.

### 5. 제어박스의 시간대를 사용자 환경에 맞게 설정하십시오.

![missing](/technical_docs/common/set_cb_time_zone/4.png)

### 6. [저장] 버튼을 클릭하십시오.

### 7. 제어박스를 재부팅하십시오.

### 8. Seu-up > Log/Utility 에서 로그데이터의 시간대가 사용자가 설정해준 제어박스의 시간대와 일차하는 것을 확인할 수 있습니다.

![missing](/technical_docs/common/set_cb_time_zone/5.png)
