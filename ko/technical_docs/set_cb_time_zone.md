# 제어박스 시간 설정하기 v6.5

### 1. 태블릿을 제어박스와 연결하십시오.

### 2. Set-up > Log 경로로 이동하십시오.

### 3. 제어박스의 시간대를 확인하십시오.

제어박스의 시간대를 확인하십시오. (시간대 설정은 로그 파일의 정확한 이름 지정 및 향후 로봇 유지보수에 영향을 미칩니다.)

![missing](/technical_docs/common/set_cb_time_zone/1.png)

### 4. System > Configuration 경로로 이동하십시오.

### 5. 제어박스의 시간대를 사용자 환경에 맞게 설정하십시오.

![missing](/technical_docs/common/set_cb_time_zone/2.png)

### 6. [저장] 버튼을 클릭하십시오.

### 7. 제어박스를 재부팅하십시오.
