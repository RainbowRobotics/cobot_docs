# 리눅스 Ui 설치하기

<div class="warning custom-block">
    <p class="custom-block-title">주의</p>
    <ol>
        <li>
            이 매뉴얼은 우분투 22.04 에서 제작 되었습니다. 
        </li>
    </ol>
</div>

1. 레인보우로보틱스 공식 홈페이지에 접속합니다. (https:/www.rainbow-robotics.com)
   
2. 상단의 '기술지원' -> '협동로봇 기술자료'에 접속합니다. 

![missing](/technical_docs/common/ui_linux/a1.png)

3. 상단의 '다운로드'를 클릭하여 원드라이브 링크에 접속합니다. 

![missing](/technical_docs/common/ui_linux/a2.png)

4. '1_Software_Release' -> '1.1_LTS_x.x.x_' -> '3_Linux' 에서 .zip 파일을 다운로드 합니다.

5. 다운로드 되어진 파일의 압축을 풀어줍니다. 

6. 마우스 오른쪽을 클릭하여 파일을 실행 시킨 후, Properties를 실행합니다. 

![missing](/technical_docs/common/ui_linux/a3.png)

7. Permission탭에서 'Allow executing file as program'을 체크합니다. 

![missing](/technical_docs/common/ui_linux/a4.png)

8. 파일을 더블클릭하여 레인보우로보틱스 Ui를 실행시킵니다.
