# Set-up (Devices)

Set up additional devices (Devices) connected to the robot.

### 1. Analog Weld Machine Setting

![missing](/manual/en/setup/13-1.png)

1. Setting can be carried out for the analog welding machine and parameters related to the welding output are set as a linear function.
2. You can save the changed settings.

### 2. Advanced Analog Welder

![missing](/manual/en/setup/13-2.png)

3. The parameters associated with the welding output can be set to the user's discretion.

### 3. Digital Weld Machine Setting

![missing](/manual/en/setup/13-3.png)

4. If you use the digital welder function in the Make window, you will immediately see what you set in the corresponding compartment.
5. You can save the changed settings.

### 4. Gripper Setting

![missing](/manual/en/setup/13-4.png)

6. If you use the gripper function in the Make window, you will see what you set up in the corresponding space immediately.
7. You can save the changed settings.

### 5. Communication I/O Map

![missing](/manual/en/setup/13-5.png)

8. You can map the device in the PLC to the I/O in the control box, and it can be used even if the program is not running.
9. You can save the changed settings.

### 6. External Axis Setting

![missing](/manual/en/setup/13-6.png)

10. Proceed with the settings for each parameter on the external axis.
11. You can save the changed settings.
